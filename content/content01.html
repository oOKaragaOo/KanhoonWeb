JS Project ที่ควรมีติด Resume

เพิ่มโอกาสในการทำงานในระดับ 1,000,000%


Photo by Canva

ถ้าคุณต้องการเรียนรู้เพิ่มเติมนอกจากเนื้อหาที่จะแสดงต่อไปนี้ สามารถติดตามในช่อง Youtube ของเจ้าของเนื้อหา มาเพิ่มความสามารถไปพร้อมกับเรากันเลย https://www.youtube.com/@litwire

ไม่ว่าคุณจะอ่านหนังสือเกี่ยวกับการเขียนโปรแกรม วิดีโอที่คุณดูหรือพอดแคสต์ที่คุณฟังกี่เล่ม คุณยังต้องฝึกฝนอย่างต่อเนื่องในขณะที่มีเฟรมเวิร์กมากมายพัฒนาอยู่ตลอดเช่น — React, Angular, Vue และ Svelte เป็นต้น และแน่นอน เฟรมเวิร์คเล่านี้ล้วนใช้ JavaScript ในการพัฒนา ในวงการ Developers ไม่ว่าจะมีประสบการณ์อยู่แล้ว หรือมือใหม่ การเรียนรู้ JavaScript, HTML และ CSS เป็นสิ่งจำเป็น อย่างมาก แต่วิธีที่ดีที่สุดในการฝึกฝน JavaScript คืออะไร? คําตอบที่ดีที่สุดอยู่ข้างหน้านี้แล้ว !

  1. App นั่งสมาธิ JavaScript Project

    2. คุณกําลังจะสร้างแอปพลิเคชันที่สามารถเลือกเสียงรอบข้างสําหรับการทําสมาธิ และเลือกระหว่างตัวจับเวลาและเสียงที่แตกต่างกัน

  2. App ธีมแป้นพิมพ์โทรศัพท์

    3. แป้นพิมพ์เสมือน เริ่มต้นด้วย Vanilla JS, HTML และ CSS ที่สามารถใช้ในเบราว์เซอร์และโทรศัพท์!

  3. สร้างตะกร้าสินค้าบนเว็ป ด้วย JavaScript

    4. คุณจะสร้างตะกร้าสินค้าที่สามารถใช้กับ ร้านค้าออนไลน์และไซต์อีคอมเมิร์ซด้วย Vanilla JS, HTML และ CSS

  4. App สภาพอากาศ

    5. สร้างแอปพลิเคชันสภาพอากาศด้วย JS, HTML และ CSS ใช้ Dark Sky API เพื่อดึงข้อมูลสภาพอากาศเรียนรู้วิธีโต้ตอบกับ API ซึ่งเป็นอีกสิ่งที่ยอดเยี่ยมที่คุณสามารถทําได้ด้วยจาวาสคริปต์

  5. Issue Tracker JavaScript Project

    6. ในคริปนี้เราจะพาคุณสร้าง issuse tracker เราจะมานำโปรเจคนี้มาใส่บนเว็ปไซต์ของเรา โดยการใช้ JS, HTML และ CSS เป็นโปรเจคที่มือใหม่เข้าใจได้ง่าย และมันต้องดีแน่นอนหาก HR เห็นโปรเจคนี้บน Resume ของเรา

  6. PIN Pad JavaScript Project

    7. เรามาสร้างแป้นแบบ pin pad กล่าวคือแป้นที่เหมือนกับการใส่รหัสเพื่อปลดล๊อคจอทรัชกรีนนั้นเอง แต่เราจะนำสิ่งนี้มาใส่ใน เว็ปไซต์ของเรา!! มันต้องสวยงามพระราม 8 แน่นอน !

  7. Landing Page JavaScript Project

    8. หลังจากผ่านโปรเจคมามากมาย พร้อมรึยังที่จะสร้างเว็ปไซต์ในเวอร์ชั่นที่สวยงามหูตาแตก ด้วยรูปแบบเว็บที่เรียกกันว่า Landing Page!!

  8. Rock Paper Scissors Game JavaScript Project

    9. มาสร้างเกมแรกจาก JavaScript ด้วยเกมยอดฮิตตลอดการ นั้นก็คือ "เกมเป่า ยิง ฉุบ" แน่นอนความสนุกเพลิดเพลินเป็นสิ่งที่ควรจะอยู่บนเว็ปไซต์ของคุณ

  9. Tic Tac Toe Game JavaScript Project

    10. มาแล้วเกมที่สองของเรา ด้วยเกมที่มีชื่อเรียกว่า Tic Tac Toe Game!! หรือเกม XO นั้นเอง โดยใช้ความรู้ JS, HTML และ CSS ที่คุณมีอยู่แล้ว และแน่นอนไม่ต้องกลัวคริปสอนอยู่ข้างหน้านี้แล้ว

  10. Build 9-in-1 Mini JavaScript Projects

    11. นี้เป็นโบนัสนิดหน่อย ผมอยากให้คุณนำความรู้ทั้งหมดในบทความนี้ นำมาผสานรวมกันเป็น 1 และมันจะกลายเป็นเว็บไซต์ที่สมบูรณ์แบบ และแน่นอนนี้ไม่ใช่เนื้อหาทั้งหมดยังมีเนื้อหาอีกมากมาย มีหลากหลายโปรเจคที่เรารวมไว้ใน YouTube Channel >> https://www.youtube.com/@litwire
